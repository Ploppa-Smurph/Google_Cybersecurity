        03) What are Networks?
                  a network is a group of connected devices
                    home network = printer, laptop, desktop, smart tv, etc...
                    office network = workstations, printer, servers, etc... 
                IP address: 
                MAC address: 
                LAN (Local Area Network): spans small area, house, office, school, etc...
                WAN (Wide Area Network): much larger area, city, state, country -=- the Internet is the largest WAN 
        
        04) Working in Network Security
                  keep network systems safe and resilient  
            always keep learning -- team sport

        05) Useful Skills for Network Security
                simulate advesaries and threats
                  - programming
                  - operational and platform security 
                  - command line = allows you to interact with low levels of the operating system like memory and higher level like applications
                  - log parsing = helps in debugging and finding root issues
                  - network traffic analysis = looking across network and application layers to see what traffic is doing
                      -Could passwords be getting leaked? 
                      -Is the infrastructure secure?
                      -Are firewalls being readily configured securely?
        
        06) Network Tools
                Hub: broadcasts to ALL devices
                Switch: broadcasts to specific device or devices, more intelligent than a hub. only passes data to intended device
                Router: connects multiple networks together
                Modem: connects router to the internet, brings internet to the LAN
                Virtualization Tools: software that perform network operations, offered by cloud service providers
